# Yvonne Hair and Makeup - Professional Website

## Overview

This repository contains the complete codebase for Yvonne's professional hair and makeup artist website. The site is built using modern web technologies including Bootstrap 5, HTML5, CSS3, and JavaScript, providing a responsive, elegant experience that showcases Yvonne's services and portfolio.

> "Elegance is not about being noticed, but about being remembered."

## Live Demo

Visit the live site at [yvonne-hairmakeup.com](https://ivonna-beauty-salon-production.up.railway.app/)

## Features

- **Responsive Design**: Fully optimized for all devices from mobile to desktop
- **Modern UI/UX**: Clean, elegant design with smooth animations and transitions
- **Service Showcase**: Detailed sections for all hair and makeup services
- **Portfolio Gallery**: Image gallery of previous client work and styles
- **Testimonials**: Client reviews and feedback section
- **Pricing Packages**: Clear presentation of service packages and pricing
- **Contact Form**: Easy-to-use contact form for booking inquiries
- **Social Media Integration**: Connected to Facebook and Instagram profiles
- **SEO Optimized**: Structured for excellent search engine visibility

## Technology Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with responsive design principles
- **Bootstrap 5**: Framework for responsive grid and components
- **JavaScript/jQuery**: Interactive elements and animations
- **Swiper.js**: Touch-enabled slider/carousel
- **Iconify**: Modern icon system

## Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/yvonne-hairmakeup.git
   ```

2. Navigate to the project directory:
   ```bash
   cd yvonne-hairmakeup
   ```

3. Open `index.html` in your browser or set up a local server:
   ```bash
   # Using Python for a simple server
   python -m http.server 8000
   ```

4. Visit `http://localhost:8000` in your web browser

## Customization Guide

### Changing Content

Most content can be modified directly in the HTML files:

- **Text**: Edit the text within HTML elements
- **Images**: Replace files in the `images` directory and update paths in HTML
- **Services**: Modify the service descriptions in the services section
- **Prices**: Update the pricing information in the pricing section

### Styling Changes

- **Colors**: Primary color scheme variables are defined at the top of `css/custom.css`
- **Fonts**: Font families can be changed in the CSS files or by adding new fonts to the `fonts` directory
- **Spacing**: Adjust padding and margin values in the CSS files

### Adding New Sections

1. Create a new section in the HTML following the existing pattern
2. Add corresponding styles in `custom.css`
3. If needed, add JavaScript functionality in `custom.js`
